# AI Data Analyst Agent

An AI-powered data analysis tool that allows users to upload a CSV dataset and ask questions in natural language.

The AI agent analyzes the dataset and generates insights automatically.

## Features

* Upload CSV datasets
* Ask questions about your data
* AI generates insights using Python and Pandas
* Interactive web interface using Streamlit

## Tech Stack

* Python
* Streamlit
* LangChain
* OpenAI
* Pandas

## How to Run Locally

1. Clone the repository

git clone https://github.com/YOUR_USERNAME/ai-data-analyst-agent.git

2. Install dependencies

pip install -r requirements.txt

3. Run the app

streamlit run app.py

## Example Questions

* Which product has the highest sales?
* What is the average revenue?
* Show top 5 rows of the dataset.
